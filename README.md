# m-index-lab

## 项目简介

`m-index-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为实验性页面集合的存放与展示空间。  
仓库中的页面内容可能涵盖各类前端实验、功能原型、工具页面或静态资料展示。

## 目录说明

```
m-index-lab/
├── index.html          # 仓库主页入口（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放各个独立的 HTML 页面文件，每个文件为一个完整的页面。
- `assets/`：存放页面共用的样式、脚本或图片资源，便于复用与维护。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开查看。
- 页面之间相互独立，无依赖关系。
- 归档内容可能随时间更新或移除，不保证长期可用性。
- 页面内容仅供学习、测试或参考使用，不代表任何正式项目或服务。

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的任意 HTML 文件。
3. 或通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
